# Contributing to Fabritorio

Thank you for wanting to contribute. That genuinely means a lot this early.

The best first step is to **open an issue** so we can talk it through together,
whether it's a bug, a question, or an idea you'd like to build.

A bit of honesty: this is my very first open-source project as a maintainer, and
I'm still figuring out how I want to open it up to contributions. What fits the
direction, how to scope work, how to review it well. So give me a week or so to
find my footing, and I'll come back with something clearer. If you've maintained
open source before and have a feel for this, I'd love for you to reach out. That
perspective would help a lot.
